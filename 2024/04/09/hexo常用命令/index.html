hexo常用命令

发表于2024-04-09|更新于2024-12-08|开发
|阅读量:

hexo基本功能使用记录

new

新建文章，layout默认为post，如果不指定layout，则默认为post。

publish

发表草稿。草稿即不显示在网页上的一些文章。

server

clean

清除缓存文件(db.json)和已生成的静态文件(public)。
在某些情况下，public文件夹中的文件没有按照预期的那样生成，可以尝试使用该命令。

deploy

将网站部署到github上，-g参数表示生成静态文件。

更多命令参考Hexo官方中文文档
